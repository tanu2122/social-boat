# SocialBoat

To develop a front end web application with user profile 

## Installation and Setup

- Run `git clone https://github.com/tanu2122/social-boat.git`

- Run `npm install `

- Run `npm run start`

## Environments and Deployments

| Environment | Base URL | Description  
| :-------:   | :------: | :----------:
